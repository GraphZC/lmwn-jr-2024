## Get started

### Install

```bash
go mod download
```

### Start Production

```bash
make start
```

### Start Development

```bash
make dev 
```

I use air reload if you do not have please run the command

```bash
go install github.com/cosmtrek/air@latest
```

### Run Test

```bash
make run-test
```
